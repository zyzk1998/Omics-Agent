# 动态技能插件编写指南（给生物 / 化学同事）

本指南说明如何准备一份可被系统**自动解析并上架到技能广场**的技能包。您**不需要会编程**即可完成 `SKILL.md`；若需要真正在计算节点上跑算法，再请同事补充 `main.py`。

---

## 一、您需要交付什么？

任选一种方式：

| 方式 | 内容 | 适用场景 |
|------|------|----------|
| **A. 完整 ZIP** | 内含 `SKILL.md`（或 `skill.yaml`）+ `main.py` | 既要说明文档，又要可执行 |
| **B. 仅 Markdown** | 单独一个 `.md` 文件上传 | 先登记参数与说明，执行代码稍后由同事补进 ZIP |

> 推荐：**优先使用 ZIP**，且根目录下直接包含 `SKILL.md` 与 `main.py`（不要多一层空文件夹，除非整个包只有一个子文件夹且 `main.py` 只在那一层里）。

---

## 二、`SKILL.md` 标准模板（复制后改字即可）

把下面整段复制到新文件 `SKILL.md`，按注释改掉即可。

```markdown
## Name

我的代谢物峰对齐工具

（系统会取这一行或本段第一行作为技能名称；英文标识会自动生成，请避免与已有技能重名。）

## Description

用于 LC-MS 代谢组数据保留时间漂移校正与峰表导出。适用于非靶向代谢流实验。

## Parameters

- input_csv (string): 输入峰表 CSV 的路径（由用户在对话里上传数据资产后，由助手填入）(required)
- rt_window (number): 保留时间匹配窗口，单位分钟，默认 0.05
- polarity (string): 离子模式，可选 positive 或 negative
```

### 参数行写法（很重要）

每一行一条参数，格式：

```text
- 参数名 (类型): 说明文字
```

支持的类型关键字（写英文即可）：

- `string` — 文本、文件路径等  
- `number` / `float` — 小数  
- `integer` / `int` — 整数  
- `boolean` / `bool` — 是/否  

若必须填写，请在同一行末尾加上 **`(required)`**（英文括号即可）。

## 三、`main.py`（需由会 Python 的同事编写）

规则只有两条：

1. 文件中必须有一个函数：**`def run(**kwargs):`**
2. 返回值必须是 **字典**（JSON 可序列化），且建议包含：
   - `status`: `"success"` 或 `"error"`
   - `message`: 人类可读说明
   - 若有结果文件，可返回 `file_path` 等键（具体与内部工具约定为准）

示例骨架：

```python
def run(**kwargs):
    input_csv = kwargs.get("input_csv")
    if not input_csv:
        return {"status": "error", "message": "缺少 input_csv"}
    # TODO: 在这里写实际计算逻辑（或调用已有脚本）
    return {"status": "success", "message": "处理完成", "output_csv": "/path/to/out.csv"}
```

**注意：** 真正耗时的计算会在独立 **worker 容器**里执行，主站只负责解压、解析与登记，不会在聊天进程里直接执行您的代码。

## 四、打包成 `.zip`

1. 确认目录里有：`SKILL.md`、`main.py`（若暂无可先只准备 `SKILL.md`，单独上传 `.md` 做说明登记）。  
2. 选中这两个文件（及依赖的其它只读数据文件，若有），**压缩为 ZIP**。  
3. 在网页技能广场点击 **上传技能**，将 ZIP **拖入虚线框**或点击选择文件上传。  
4. 上传成功后，在「全部」列表靠前位置会出现 **大类为「动态插件」** 的卡片；若筛选大类为 **「动态插件」** 也可只看到这类技能。

---

## 五、环境与安全（Env）

- 不要在文档或代码里写**密码、Token、私钥**。  
- 需要特定软件环境时，请在 `## Description` 里用文字写清依赖（例如：需要 R 4.3、某某包版本），由平台侧评估是否纳入 worker 镜像。  
- 压缩包内**不要**包含可执行二进制安装包（如 `.exe`），除非经平台安全评审。

---

## 六、常见问题

**Q：我只会上传表格和说明，不会写 `main.py` 怎么办？**  
A：可以先只上传写好的 `SKILL.md`（单独 `.md` 文件）。平台会先登记参数结构；等同事补全 `main.py` 后再上传完整 ZIP 覆盖同名技能（需同注册策略，具体以平台提示为准）。

**Q：中文名称和英文标识有什么区别？**  
A：`## Name` 下展示给同事看的中文（或全称）；系统会生成英文 `name` 标识用于内部路由，重名时会提示更换。
