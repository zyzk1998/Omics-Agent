历史版本记录 (Changelog)

以下为按次版本线（如 V0.14、V0.13）汇总的更新摘要：同一发行线（0.14.x、0.13.x）内的补丁合并叙述，不再为每个补丁号单独开一节。更早细节请查阅仓库 gibh-desktop-app/客户端版本迭代.md 或历史提交。

V0.14.5（当前推荐）
  • 【V0.14 发行线补丁】客户端 semver 0.14.5；Windows NSIS（Omics Agent Setup 0.14.5.exe）与 Linux AppImage（Omics Agent-0.14.5.AppImage）同步至本站 /downloads/；自动更新元数据与安装包同源构建。
  • Web 端：头像菜单恢复「下载客户端」弹窗，版本与安装包链接与下载页 client-release.json 同步。
  • 内置网页 / 后端技能：第三批生物医药科学计算（ITC、细胞周期、酶/蛋白酶动力学、序列保守性）及蛋白同源结构评估、抗体人源性（OASis）等能力接入技能广场。
V0.14.4
  • 【V0.14 发行线补丁】客户端 semver 0.14.4；Windows / Linux 双端同步；自动更新源与网页同源；版本中心 Modal。
V0.14.3
  • 【V0.14 发行线补丁】客户端 semver 0.14.3；Windows / Linux 双端同步；版本中心与右侧栏工作台互斥等。
V0.14.2
  • 【V0.14 发行线补丁】客户端 semver 0.14.2；Windows NSIS 与 Linux AppImage 同步至本站 /downloads/。
  • 桌面：系统托盘与单实例锁、关闭隐藏到托盘、双击托盘唤起主窗口（需有效托盘图标文件）。
  • 网页：拖拽上传 dragCounter 防抖；工作流卡片文件并入全局输入区预览；Toast/Modal 长路径换行；问题反馈闭环等。
  • 自动更新：元数据与安装包同源构建。
V0.14.0
  • 【V0.14 发行线】客户端 semver 0.14.0；Windows NSIS（Omics Agent Setup 0.14.0.exe）与 Linux AppImage（Omics Agent-0.14.0.AppImage）已同步至本站 /downloads/。
  • 自动更新：latest.yml、latest-linux.yml 与对应 .blockmap 与安装包同源构建，electron-updater（generic）可按哈希校验增量。
  • 内置网页：与主仓库同步；含 Markdown 表格渲染修复（诊断区「参数推荐」等表格正常展示）、流式与排版类改进（以当前打包内 bundled-web-ui 为准）。
  • 运维提示：若在本机构建新平台包，请在对应环境执行 npm run dist:win / dist:linux 后务必再运行 npm run sync:download-page，避免 sha512 与文件名漂移。
V0.13.x（系列汇总）
  • Sidecar / 8019：本地 FastAPI 中间件、POST /api/shutdown、进程树清理、端口强制夺回（forceReclaimSidecarPortSync）；NSIS installer.nsh 安装前后 taskkill local_sidecar.exe。
  • 本地附件与路径：check_file POST/GET 兼容、上传云端、三态路径解析、沙盒与大文件分流、/api/chat 载荷中 local_files 与已上传路径去重。
  • 交互：右侧栏折叠与可调宽、实体附件卡片、欢迎页与文件树检索拖拽等（补丁迭代均归入本条目不逐一列出）。
  • 发版纪律：可执行内容变更须递增 package.json 版本并重打对应平台包，否则 latest*.yml 无法区分新二进制。
V0.12.x（系列汇总）
  • Sidecar 引入与打包：127.0.0.1:8019 本地轨、双轨读写、PyInstaller 侧车按平台打入 extraResources（曾修复 Windows 误带 Linux 二进制导致体积膨胀等问题）。
  • 桌面能力：挂载本地工作区、文件树与一键引用、Flexbox 布局与安全路径约束；工作台报告与网页版式对齐、RDKit 等能力与后端对齐。
  • 更小补丁（0.12.1～0.12.9）中的稳定性与元数据同步细节合并至此，不再分号单独展开。
