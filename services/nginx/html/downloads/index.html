下载桌面客户端

安装后将从独立窗口打开本站的 Omics Agent 界面（瘦客户端）。当前推荐 V0.11（构建版本号 0.11.0）。请根据您的系统选择安装包。

Windows（64 位）

NSIS 安装程序，下载后双击按向导安装；安装后可在开始菜单或桌面启动。

下载 Windows 安装包

管理员尚未将安装包放到本站 /downloads/ 目录。请将构建得到的 .exe（见下方常见文件名）复制到该目录后刷新本页。

Linux（x86_64）

AppImage（在 gibh-desktop-app/package.json 的 build.linux.target 中配置）。下载后执行 chmod +x *.AppImage，再双击或在终端运行。

下载 Linux AppImage

尚未检测到 AppImage。请在 Linux 上进入 gibh-desktop-app 执行 npm run dist:linux（或 npm run dist -- --linux），将 dist-out 内生成的 .AppImage 复制到服务器仓库的 services/nginx/html/downloads/ 后刷新本页（勿写到 dist-out 当作下载目录）。

说明：客户端内嵌的服务地址在构建时写入 main.js（当前为 loadURL 指向的地址）。更换公网或内网入口时请重新打包后再上传安装包。
