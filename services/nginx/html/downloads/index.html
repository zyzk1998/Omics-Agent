历史版本记录 (Changelog)

以下为近期版本摘要；更早版本请查阅仓库或发行说明。

V0.13.3（当前推荐）
  • 【V0.13.3 · check_file 对接与兼容】
  • Sidecar 增加 POST /api/tools/check_file（JSON 传 path）；前端对称支持；旧 Sidecar 仅支持 GET 时自动回退 GET+encodeURIComponent。
  • 本地闸门称重逻辑兼容重启后旧二进制，无需强制升级即可恢复；仍建议升级以获得 POST 路径与最新 Sidecar。
V0.13.2
  • 【V0.13.2 · 路径与本地数据分流】
  • 三态路径解析、Sidecar 上传云端、沙盒弹窗与 <200MB 静默同步等（详见上一版说明）。
V0.13.1
  • 【V0.13.1 · 交互与布局精修】
  • 右侧侧边栏可折叠、可调宽；实体附件卡片与欢迎页排版精修（详见历史说明）。
V0.12.9
  • 【V0.12.9 · Bug 修复】
  • 修复 Windows 安装包误将 Linux 版 Sidecar 一并打入 extraResources，导致安装包体积异常膨胀（约多出七十余兆字节）的问题；现按平台仅打入对应二进制（Windows：local_sidecar.exe；Linux：同名无后缀可执行文件）。
  • 下载页与自动更新元数据同步为本版本瘦身后的 Windows / Linux 安装包。
V0.12.8
  • 【V0.12.8】
  • 内置 PyInstaller Sidecar，免用户机 Python；npm run dist:* 链式构建。
  • 补充按平台拆分 Sidecar 资源说明（随后发现 Windows 包仍误带 Linux 二进制，已于 V0.12.9 修复）。
V0.12.7
  • 【V0.12.7】
  • 生产包通过 extraResources 携带 Sidecar；process.resourcesPath 定位资源。
  • 开发态仍可用系统 Python 调试；生产态逐步切换为内置二进制（见 V0.12.8）。
  • 自动更新与挂载工作区相关体验优化见该版本发版说明。
V0.12.6
  • 【V0.12.6】
  • 新增 Local Sidecar 本地中间件（127.0.0.1:8019），将本地工作区挂载与文件树能力下沉到宿主机执行。
  • 新增本地中继工具接口：/api/tools/read_file 与 /api/tools/write_file，支持前端桥接本地文件读写。
  • 新增 Dual-Track 双轨机制：Web 轨道保持云端 /app/uploads、/app/results 传统链路；本地轨道走前端 relay + Sidecar。
  • 优化 Electron 生命周期：应用启动自动拉起 Sidecar，退出时自动回收 Python 子进程。
V0.12.3
  • 【V0.12.3 核心更新】
  • 新增：桌面级高权限模式，支持挂载本地项目工作区。
  • 新增：右侧资源管理器 (File Explorer)，实时映射本地代码/数据目录。
  • 新增：文件「一键引用」功能，点击文件树节点即可快速加入对话上下文。
  • 优化：重构主界面 Flexbox 弹性布局，自适应侧边栏收缩，消除 UI 塌陷隐患。
  • 安全：增加目录穿越 (Path Traversal) 物理防护墙，锁定智能体读写范围。
V0.12.2
  • Windows 安装包：工作台报告打印/导出与网页版版式对齐，修复整页空白等问题。
  • 化学信息学（RDKit）相关工具与技能能力对齐服务端。
  • 编排与对话：会话标题、工具输出记忆等体验优化；技能广场与前端若干改进。
V0.12.1
  • （占位）桌面端稳定性与网络重试策略优化。
  • （占位）与 Web 端 API 协议小版本对齐。
