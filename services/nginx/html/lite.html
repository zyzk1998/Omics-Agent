🚀 Omics Agent - Thin Client

轻量级前端演示 - 展示如何与后端 API 集成

📤 1. 文件上传 API

上传文件到 POST /api/upload，支持多文件上传和 10x Genomics 数据自动识别

💬 2. 聊天 API

发送消息到 POST /api/chat，支持流式响应（SSE）和 JSON 响应

📚 API 接口说明

1. 文件上传接口

  • Endpoint: POST /api/upload
  • Content-Type: multipart/form-data
  • 请求参数: files (File[]), user_id (可选), session_id (可选)
  • 响应格式: JSON
  • 响应字段: status, file_paths (数组), file_info (数组), user_id, session_id
  • 特殊处理: 自动识别 10x Genomics 数据（matrix.mtx, barcodes.tsv, features.tsv）并分组保存

2. 聊天接口

  • Endpoint: POST /api/chat
  • Content-Type: application/json
  • 请求体字段:
    • message (string): 用户消息
    • uploaded_files (array): 已上传的文件列表，格式: [{"name": "file.csv", "path": "path/to/file.csv"}]
    • stream (boolean): 是否使用流式响应，默认 false
    • user_id (string, 可选): 用户ID，默认 "guest"
    • session_id (string, 可选): 会话ID，未提供时自动生成
    • history (array, 可选): 对话历史
    • workflow_data (object, 可选): 工作流执行数据
  • 响应类型:
    • 流式响应 (SSE): text/event-stream，事件类型包括: status, message, workflow, step_result, diagnosis, result, done, error
    • JSON 响应: application/json，包含 type 字段（如 workflow_config, analysis_report）
