## 四联同步（变更任一处须核对其余三处）

| 链路 | 约定 |
| --- | --- |
| **`thought` SSE** | 写入快照 **`reasoning`**，不得混入 `text` |
| **`message` SSE** | 写入快照 **`text`** |
| **`status` SSE**（非 `[AgenticLog]`） | 追加 **`process_log`** 条目 `{content, state}`；**`done`** 须写 **`duration`** |
