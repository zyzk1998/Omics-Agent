# 工作流 API

工作流 API 提供工作流规划、保存、列表、删除等功能。

## 接口列表

1. [规划工作流](plan.md) - `POST /api/workflows/plan`
2. [保存工作流](save.md) - `POST /api/workflows/save`
3. [列出工作流](list.md) - `GET /api/workflows/list`
4. [删除工作流](delete.md) - `DELETE /api/workflows/{workflow_id}`
5. [任务历史](jobs-history.md) - `GET /api/jobs/history`
6. [工作流状态](status.md) - `GET /api/workflow/status/{run_id}`
